#### ODULE NAME
       Flopenr

#### DEPENDENCIES
       DESwitch

#### INPUTS
Name  | Size(Bits)
------|------------
clk  |     1      
reset |     1      
D   |     1      
E   |     1      

#### OUTPUTS
Name | Size(Bits)
-----|------------
Q   |     1      

#### DESCRIPTION
       A  one-bit  enabled  register, implemented with two switches (one unen-
       abled, one enabled)
